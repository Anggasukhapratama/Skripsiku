# 🎓 IngatSkripsiku

Dashboard interaktif dan *tracker* produktivitas pribadi untuk manajemen jadwal, target bimbingan, dan penyelesaian Skripsi. Dibangun sepenuhnya menggunakan antarmuka yang bersih dan responsif.

## ✨ Fitur Utama
- **Countdown Sidang Akhir:** Hitung mundur menuju *deadline* pengumpulan atau jadwal sidang.
- **Bimbingan Tracker:** Modul khusus untuk melacak jumlah progres bimbingan dengan Dosen Penguji 1 & 2.
- **Micro-Tasking (Checklist Bab):** Memecah pengerjaan dokumen atau eksekusi sistem ke dalam sub-bab kecil agar tidak *overwhelming*.
- **Alarm Pengingat Otomatis:** Sistem notifikasi browser harian untuk membangun rutinitas pengerjaan skripsi.
- **Persistent Storage:** Menggunakan *Local Storage* API sehingga semua input dan riwayat tugas aman tersimpan di *browser* tanpa perlu *database* eksternal.

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## 🚀 Cara Menjalankan
Aplikasi ini bersifat *client-side* seutuhnya. Tidak memerlukan *setup backend* khusus seperti Flask atau Node.js.
1. Clone repositori ini atau *download* ZIP.
2. Buka file `skripsi-dashboard.html` langsung di *browser* pilihan Anda.
